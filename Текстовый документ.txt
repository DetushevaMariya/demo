# 1. Сервер
cd server
npm init -y
npm i express pg cors dotenv jsonwebtoken
node src/index.js

# 2. Клиент (в новом терминале)
cd client
npm create vite@5.4.0 . -- --template react
npm install
npm i react-router-dom
npm run dev

Чтобы перенести скрипт из init.sql в Postgress, пишем команду в терминале:
& "C:\Program Files\PostgreSQL\(версия)\bin\psql.exe" -U postgres -c "CREATE DATABASE (название бд);"
Затем:
cd server
& "C:\Program Files\PostgreSQL\(версия)\bin\psql.exe" -U postgres -d (название бд) -f init.sql

ER-диаграмма: В pgAdmin или DBeaver кликните правой кнопкой по схеме → Generate ER Diagram. Свяжите users.id → applications.user_id → reviews.application_id.

Финальный запуск:
# 1. Сервер (в отдельном терминале)
cd server
node src/index.js

# 2. Клиент (в новом терминале)
cd client
npm run dev
